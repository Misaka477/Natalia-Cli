# Natalia CLI Help

回答关于 Natalia CLI 用法的问题。

## 命令
- /setup - 配置 LLM
- /profile - 切换配置
- /rule - 切换规则
- /rollback <n> - 回退到第 n 步
- /sandbox - 管理沙盒
- /workers - 查看子 agent
- /help - 帮助

## 规则
- code - 编程
- ask - 问答
- plan - 规划
- debug - 调试
- chat - 聊天
